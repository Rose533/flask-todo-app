# 📋 Flask ToDo App

這是一個使用 Python Flask 框架開發的個人待辦清單系統，使用 SQLite 資料庫儲存資料，支援任務新增、刪除，並成功部署到雲端平台 Render。

---

## 🔧 使用技術

- Python 3
- Flask
- SQLite
- Jinja2（模板語言）
- Git + GitHub
- Render 雲端部署

---

## ✅ 功能介紹

- 新增任務
- 刪除任務
- 雲端部署，全球可使用
- 免費平台 Render 自動部署
